# capcopy

A Chrome extension to copy Google Meet captions to clipboard.

## Installation

1. Run `npm install && npm run build`
2. Open `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the `dist` folder

## Usage

On a Google Meet page with captions enabled:

- **Keyboard shortcut**: `Ctrl+Shift+Y` (Mac: `Cmd+Shift+Y`)
- **Extension menu**: Click the extension icon and press "Copy Captions"

## Development

```bash
npm install     # Install dependencies
npm run build   # Build the extension
npm run watch   # Watch mode for development
```
